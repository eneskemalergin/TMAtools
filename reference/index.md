# Package index

## Main function

Main function which runs the entire TMAtools pipeline.

- [`tmatools()`](https://edgeresearch-ca.github.io/tmatools/reference/tmatools.md)
  : TMAtools pipeline

## Individual steps

Each major step in the TMAtools pipeline.

- [`combine_tma_spreadsheets()`](https://edgeresearch-ca.github.io/tmatools/reference/combine_tma_spreadsheets.md)
  : Combine TMA spreadsheets
- [`deconvolute()`](https://edgeresearch-ca.github.io/tmatools/reference/deconvolute.md)
  : Deconvolute a combined TMA spreadsheet
- [`translate_scores()`](https://edgeresearch-ca.github.io/tmatools/reference/translate_scores.md)
  : Translate biomarker scores from a deconvoluted TMA spreadsheet.
- [`consolidate_scores()`](https://edgeresearch-ca.github.io/tmatools/reference/consolidate_scores.md)
  : Consolidate biomarkers from a translated TMA spreadsheet.

## Post-consolidation analysis

Functions for downstream analysis of consolidated TMA data.

- [`assess_mmr_status()`](https://edgeresearch-ca.github.io/tmatools/reference/assess_mmr_status.md)
  : Assess MMR status based on IHC biomarker scores
